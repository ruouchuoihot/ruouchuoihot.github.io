# **2. Các thành phần Docker Objects**

Docker Engine quản lý nhiều loại Docker Objects khác nhau, bao gồm:

## **Volumes**

- Volumes lưu trữ dữ liệu lâu dài cho containers.
- Hữu ích trong việc chia sẻ dữ liệu giữa các container hoặc bảo toàn dữ liệu khi container bị xóa.
## **Networks**

- Docker Engine cung cấp nhiều tùy chọn mạng (bridge, overlay, host) để kết nối containers với nhau hoặc với hệ thống bên ngoài.
- Đảm bảo tính linh hoạt và bảo mật khi triển khai ứng dụng.
